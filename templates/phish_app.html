How to save an email

Choose your mail app, save the message, then upload that file here.

Gmail
  1. Open the email.
  2. Select More, then Download message.
  3. Upload the downloaded .eml file.
Outlook desktop
  1. Open the email in its own window.
  2. Choose File, then Save As.
  3. Save as Outlook Message Format or .eml if available.
Outlook web
  1. Open the email.
  2. Select More actions, then Download.
  3. Upload the downloaded message file.
Apple Mail
  1. Select the email.
  2. Choose File, then Save As.
  3. Use Raw Message Source or .eml when available.
Zoho Mail
  1. Open the email.
  2. Select More actions, then Export or Download.
  3. Upload the saved .eml file.
